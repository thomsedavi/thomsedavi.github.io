David's page to learn about web design and web apps and such

Project 1: Lego head finder

Have you ever wanted to source a specific Lego head, for example one with an eyepatch and stubble, but didn't want to search through the 1,000+ Lego heads in the database? This tool should make it easier for you.

Project 2: Learn a language by teaching a robot language.

Rather more ambitious! A kind of reverse language learning idea, where you learn by teaching. A robot will ask, 'what's the French word for Car', for example, and you do the research yourself. Later on you quiz the robot and correct it when it suggests that Papillon means Car. Like I said, ambitious.
